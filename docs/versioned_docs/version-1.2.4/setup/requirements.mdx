# Requirements

## System Requirements

- **Operating System**: Linux, macOS, or Windows
  - (Anything that can run Docker)
- **Architecture**: x86_64
- **Memory**: 2GB or more

:::info
ARM support is planed for a future release.
:::
## Software Requirements

- **Docker**: [Install Docker](https://docs.docker.com/get-docker/)
  - Any version of Docker should work, but we recommend using the latest version.
- **Docker Compose**: [Install Docker Compose](https://docs.docker.com/compose/install/)

:::warning
Any reasonably modern computer should be able to run Booklify. However, we wont be able to provide support for systems that don't meet the above requirements.
:::
