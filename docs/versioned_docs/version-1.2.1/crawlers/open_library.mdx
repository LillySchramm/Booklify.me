# Open Library

## Overview

**Usable by default:** Yes\
**Crawls metadata:** Yes\
**Crawls covers:** Yes\
**Overall quality:** Excellent\
**Overall quantity:** Okayish for english books, poor for other languages\
**Longrunning:** No

## Description

Open Library is a project of the Internet Archive, a nonprofit organization.
It has a goal of creating a web page for every book ever published.
It provides access to many public domain and out-of-print books, and allows users to contribute books.
It also provides a [RESTful API](https://openlibrary.org/developers/api) for developers to access the data.

## Configuration

This source doesn't need any configuration. It's usable by default.
